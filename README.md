# JourneyNest 🌍✈️

JourneyNest is a Travel Planning Web Application designed to help users explore destinations, plan trips, and manage travel information through an interactive web interface.

## 🚀 Live Demo
https://journeynest-rpro.onrender.com

## 💻 GitHub Repository
https://github.com/aditiyadav0413-creator/JourneyNest

---

## 📌 Features

- 🌍 Explore different travel destinations
- 🗺️ Plan and manage trips easily
- 🖥️ User-friendly web interface
- 📱 Responsive design
- 🔍 Easy navigation between pages
- 📊 Manage travel-related information

## ⚙️ Installation & Setup

1. Clone the repository:


git clone https://github.com/aditiyadav0413-creator/JourneyNest.git


2. Install required packages:


pip install -r requirements.txt


3. Run the application:


python app.py


4. Open your browser:


http://127.0.0.1:5000

## 🎯 Project Objective

The main objective of JourneyNest is to provide a simple and interactive platform for users to explore destinations and organize their travel plans efficiently.
